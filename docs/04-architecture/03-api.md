# Сущности и API

1. **ProductModel** - модель продукта
    - **id** - идентификатор
    - **name** - наименование
    - **description** - описание
    - **productGroupId** - ссылка на категорию продукта

   | Доступность: | Всем                        | Всем поставщикам | Владельцу            |
   |--------------|-----------------------------|------------------|----------------------|
   | Методы:      | `getById`<br>`findByFilter` | `create`         | `update`<br>`delete` |
